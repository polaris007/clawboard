## ⚠️ 不应修改的地方（确认保留）

### IssueDetector.java 中的原因分析
以下文字**应该保持为"问题"**：
- [ ] "编码问题"
- [ ] "hosts文件配置问题"
- [ ] "SSL证书问题"

这些是描述性的技术术语，不是指检测到的错误。

## 🎯 最终确认

### 术语使用场景总结

| 场景 | 应使用 | 示例 |
|------|--------|------|
| 检测到的异常 | **错误** | "发现错误150个" |
| 报告标题 | **错误** | "综合错误检测报告" |
| 统计项 | **错误** | "总错误数"、"错误率" |
| 错误编号 | **错误** | "错误 #1" |
| 错误类型表格 | **错误** | "错误类型分布" |
| 原因描述中的技术问题 | **问题** | "编码问题"、"配置问题" |
